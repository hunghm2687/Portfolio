## 💻 DỰ ÁN NỔI BẬT

### 🔨 Hệ thống Đấu giá Trực tuyến (Auction System) | *Bài tập lớn Nhóm*
> **Vai trò:** `Core Developer (Backend & Database Engineer)` – Thiết kế kiến trúc Server, lập trình logic xử lý phía Backend và thiết kế toàn bộ cấu trúc cơ sở dữ liệu hệ thống.

*   **Công nghệ:** `Java Socket`, `Multi-threading`, `MySQL`, `Git`, `Maven`.
*   **Mô tả dự án (Chuẩn STAR):**
    *   📦 **Context:** Hệ thống đấu giá trực tuyến real-time, cho phép nhiều Client kết nối đồng thời qua một Server trung tâm.
    *   🎯 **Task:** Thiết kế kiến trúc Client-Server chịu tải tốt, logic đấu giá chính xác đến từng mili-giây và đồng bộ dữ liệu xuống database an toàn.
    *   ⚡ **Action:**
        *   Áp dụng kiến trúc **Multi-threaded Socket Server** để xử lý mỗi client trên một luồng riêng biệt.
        *   Giải quyết xung đột đồng thời (**Concurrency**) bằng kỹ thuật khóa dữ liệu trong Java khi nhiều người cùng đấu giá một mặt hàng tại một thời điểm.
        *   Sử dụng **MySQL Workbench** thiết kế cơ sở dữ liệu chuẩn hóa; tối ưu hóa truy vấn SQL để cập nhật trạng thái phòng đấu giá real-time.
    *   🎉 **Result:** Hệ thống chạy mượt mà, giả lập thành công hàng chục client tương tác cùng lúc mà không bị nghẽn hay sai lệch dữ liệu.
